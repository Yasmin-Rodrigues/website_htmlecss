CRIAÇÃO DE WEBSITE EM HTML5 E CSS3

O que é HTML?

O HTML é a linguagem base para se desenvolver qualquer site baseado em marcações, onde marcamos os elementos para definir quais informações a página vai exibir. A linguagem foi criada na década de 1990 e suas especificações são controladas pela W3C (World Wide Web Consortium)

O que é CSS?

O CSS é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML, ou seja, definir a aparência das páginas, para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais.
